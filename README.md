# VNTracker

ビジュアルノベルのプレイ時間を自動計測・管理するツールです。

## 主な機能
- ゲームのプレイ時間を自動計測（フォアグラウンド検出）
- グラフ表示（日別・月別推移・集計円グラフ）
- 評価管理（カスタムカテゴリ・総合点自動計算）
- 批評空間連携（マスタDL・評価転記・積みゲーインポート）
- Etupirkaからのデータ移行
- 自動バックアップ（7日ごと・5世代保持）
- ダーク/ライトテーマ切替

## ダウンロード
[最新版はこちら](https://github.com/tamamasa0718-wq/VNTracker/releases/latest)

## インストール方法
1. ZIPファイルをダウンロード
2. 任意の場所に解凍
3. `VNTracker.exe` を実行

## 動作環境
- Windows 10/11 (64bit)
- .NETランタイム不要（同梱済み）
